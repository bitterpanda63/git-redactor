# git-redactor

> **Warning**
> this repo isn't tested, and does force pushes & rewrites, use at your own risk.

## Running
Make sure git-filter-repo is installed
```shell
brew install git-filter-repo
```

Install dependencies
```shell
npm install
```

Run (interactive)
```shell
node index.js
```
